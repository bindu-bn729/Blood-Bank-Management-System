# Blood Bank Management System

This project is a web-based application developed to manage blood donors and blood stock in a blood bank.

## Features
- Add donor details
- View donor list
- Manage blood stock
- Issue blood to patients
- User login and registration
- Dashboard for blood bank management

## Technologies Used
- Java
- Spring Boot
- HTML
- CSS
- JavaScript
- MySQL
- Maven

## Project Structure
- Controllers – Handle HTTP requests
- Repositories – Database operations
- Models – Entity classes
- HTML Pages – User interface

## How to Run
1. Clone the repository
2. Open in IntelliJ / Eclipse
3. Configure MySQL database
4. Run DemoApplication.java
5. Open browser and access the application
